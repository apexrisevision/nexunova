## 🚀 Deployment

### GitHub Pages
1. Push this folder to a GitHub repository
2. Go to **Settings → Pages → Source** → select `main` branch / root
3. Site will be live at `https://<username>.github.io/<repo>/`

### Vercel (Recommended)
1. Push to GitHub
2. Import the repository in [vercel.com/new](https://vercel.com/new)
3. Framework: **Other** (Static)
4. Root directory: `/` (or whichever folder contains `index.html`)
5. Deploy — site goes live instantly with a `*.vercel.app` URL
